# Security

FileSwipe runs **only on your Mac**. It does not upload your files anywhere.

## What the app can access

- Folders **you** choose with the system folder picker
- Your **Downloads** folder when you click Open Downloads and allow access

## What delete does

Files are moved to the **macOS Trash**, not permanently erased by the app.

## Reporting a vulnerability

If you find a security issue, please open a private GitHub security advisory on this repository, or email the maintainer via GitHub profile contact. Please do not post exploit details in public issues.
